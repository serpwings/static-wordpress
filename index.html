Static WordPress¶

A Python Library to prepare and deploy a static version of a WordPress Installation on Netlify (Static Hosting Service Provider).

How to Use static-wordpress?¶

Please check our detailed tutorial (video/text) on Static WordPress Tutorial to understand how it works. This package has a live documentation file on documentation link.
